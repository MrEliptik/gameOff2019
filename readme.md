## How to play

### Keyboard

- *Q* : Left
- *D* : Right
- *Space* : Jump (double jump)
- *Shift* : Dash
- *Ctrl* : Slow-mo

### XBox controller

- *D-pad left* or *Left analog stick to the left* : Left
- *D-pad right* or *Left analog stick to the right* : Right
- *A* : Jump (double jump)
- *X* : Dash
- *B* : Slow-mo
